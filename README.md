## Доступные игры

### 1. Brain Games - Приветствие
Простое приветствие и знакомство с игроком.

### 2. Brain Even - Проверка на четность
Определите, является ли число четным или нечетным.

### 3. Brain Calc - Калькулятор
Решите простое арифметическое выражение.

### 4. Brain GCD - Наибольший общий делитель
Найдите наибольший общий делитель двух чисел.

### 5. Brain Progression - Арифметическая прогрессия
Найдите пропущенное число в прогрессии.

### 6. Brain Prime - Простое число
Определите, является ли число простым.


## Установка и запуск

### 1. Клонируйте репозиторий
git clone https://github.com/julverik/python-project-49.git
cd python-project-49

### 2. Установите зависимости
make install

### 3. Соберите пакет
make build

### 4. Установите пакет системно
make package-install

## Запуск установленных игр
После установки команды доступны из любой директории:

### Игры
brain-games        # Приветствие
brain-even         # Проверка на четность
brain-calc         # Калькулятор
brain-gcd          # Наибольший общий делитель
brain-progression  # Арифметическая прогрессия
brain-prime        # Простое число
